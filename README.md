# 🍕 Sistema de Pizzaria (Microsoft Access)

## 🧠 Descrição

Sistema desenvolvido em Microsoft Access para gerenciamento de pedidos e clientes de uma pizzaria.

## 🎯 Objetivo

Praticar modelagem de banco de dados e criação de aplicações desktop.

## 🛠️ Tecnologias

* Microsoft Access

## ⚙️ Funcionalidades

* Cadastro de clientes
* Registro de pedidos
* Organização de dados

## 🧠 Conceitos aplicados

* Banco de dados relacional
* Formulários e consultas
* Organização de informações

## ▶️ Como executar

Abrir o arquivo no Microsoft Access.
